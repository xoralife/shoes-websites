## ✨ Features
- Modern glass-morphism UI design
- Fully responsive (Mobile, Tablet, Desktop)
- Add to cart functionality
- Wishlist feature
- Countdown timer
- Category filtering
- Testimonials section

## 🛠️ Technologies Used
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons
- Google Fonts (Inter)
